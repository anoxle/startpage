# About
A minimalist, highly customizable startpage.

## 🚀 Usage
To use this as your default new tab page, install one of the following extensions and point it to the live URL:

**Live Page:** [https://anoxle.github.io/startpage/](https://anoxle.github.io/startpage/)

### ✨ Extensions
* **Chrome / Brave / Edge / etc:** Use [New Tab Redirect](https://chromewebstore.google.com/detail/new-tab-redirect/icpgjfneehieebagbmdbhnlpiopdcmna)
* **Firefox / Zen / Waterfox / etc:** Use [New Tab Override](https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/)
